## Databricks Basics

This section covers how Databricks is used in real-world data engineering projects.

### What is Databricks?
:contentReference[oaicite:0]{index=0} is a cloud-based data platform built on Apache Spark that helps teams run ETL pipelines, analytics, and machine learning in one place.

### Core Concepts
- Workspace & Notebooks
- Clusters (managed Spark)
- Delta Lake
- Jobs & Scheduling

### Typical Workflow
1. Ingest data (CSV / Parquet / Cloud storage)
2. Transform using Spark DataFrames
3. Store in Delta tables
4. Run jobs on clusters
